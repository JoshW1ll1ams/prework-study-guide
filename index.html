HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • The h1 element is for inserting a text header, the 1 can be changed to affect the size of the element.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage. The img element can contain various attributes; the src=” ” attribute is the most important because it defines the location of the image.
  • the a element (which stands for anchor) creates links to the same webpage or other webpages. The href= “ ” attribute points to the URL for the link
  • The header appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • A footer element. The footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
  • section elements are semantic elements that define a section in a document that contains thematic content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To apply css styles to all elements on a page we use the * symbol instead of the element name or class.
  • We can apply styles to multiple elements with the same style block by separating with a commas, for example: header,footer {color: blue;}
  • To add our style sheet to our html page we link it: link rel="stylesheet" href="./assets/style.css". Rel specifies the relationship between the current document and the linked document. Href specifies the url location, in this case its a document location.
  • To apply styles by class rather than by element type we use .classname {color: blue;}

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A to add changes made to the branch once code has been saved
  • git commit -m “description” to commit our changes
  • git pull origin main to pull the source code from github to check we have latest commit
  • git push origin branchname to push the updates to github and send a pull request
  • git checkout main to get to main git branch

JavaScript

  • To add javascript first create a scripts folder in your project file then create a main.js file.
  • To import the js file to your webpage we use a script element then specify the link to the script file with src attribute
  • To declare a variable in JS we use var followed by the variable name finished with a semicolon, var myvariable;
  • We can comment using // followed by our comment.
  • Control flow refers to the order in which a computer executes code.
  • Arrays can store multiple data types
